# Seal

Basic NES games emulator using SFML graphics library.

Dependencies:
- SFML 2.6.1

Controls:

| Your PC     | Emulator |
|-------------|----------|
| X           | A        |
| Z           | B        |
| A           | Select   |
| S           | Start    |
| Up arrow    | Up       |
| Down arrow  | Down     |
| Left arrow  | Left     |
| Right arrow | Right    |
